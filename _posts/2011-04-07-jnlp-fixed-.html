--- title: JNLP fixed ! layout: post published: true --- A special thank you to Apache Felix developpement team wich has fixed the bug introduced by the update number 1.6.0_24  of java virtual machine. Use of OSGi in Java Web Start and Applet is now ok. As a result the last Kevoree (1.1.0-beta1) runtime can now be launched by the JNLP link.
